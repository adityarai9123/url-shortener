# 🔗 URL Shortener Backend

A scalable URL shortener service built using Node.js, Express, MongoDB, and deployed on Render.

---

## 🚀 Live Demo

👉 https://url-shortener-1jfb.onrender.com

---

## 📌 Features

- 🔗 Shorten long URLs into unique short links
- 🔁 Redirect short URLs to original URLs
- 📊 Track number of clicks (analytics)
- 🌐 Fully deployed backend (Render)
- 🔐 Environment variables for security

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB (Mongoose)
- NanoID (for unique IDs)
- Render (deployment)
